# 3급 보안 주입툴 (jkd_security3_injector1.exe)

대상 파일(SCF)에 SCB를 활성화 시키는, 공개용 3급 보안툴이다.</br>
기업용인 `2급 보안 주입툴`은 `유효기간`을 반드시 지정해야 하기에, 무기한 사용이 가능한 `3급 보안`을 적용시켜 공개한 것이다.</br>
홍보용인 `3급 보안 주입툴`은 보안등급 4급 이상을 주입하여, SCB를 활성화시킬 수 있다.</br>
자체검증 예제 데모 프로그램인, [selfer1_example](../selfer1_example/) 결과물(파일)에, 본 주입툴로 SCB를 활성화 시킬 수 있다.
> 명칭
  - 주입툴
> 내장 기능
  - 자체 SCB 검증
  - 대상 SCB 주입
  - 대상 SCB 검증 (간단 검증)
> 특징
  - 자신을 제외한 모든 SCF에, 4급 이상의 SCB를 주입시킬 수 있다.
  - 이미 주입된 대상 파일(SCF)은 검증을 수행하여 결과를 출력시킨다.


## 캡쳐 화면들

### 자체 검증 화면
패스워드 없이, 무기한 사용으로 설정된 홍보용 버전이다.</br>
```bash
$ jkd_security3_injector1
```
![자체 검증 화면](images/jkd_security3_injector1.png)

### 대상 주입 화면
패스워드 지정, 사용횟수 1회로 주입 시키는 장면이다.</br>
```bash
$ jkd_security3_injector1 selfer1_example.exe
```
![대상 주입 화면](images/jkd_security3_injector1-tgtinj.png)

### 대상 검증 화면
패스워드 지정, 사용횟수 1회로 주입된 대상을 검증한 화면이다.</br>
```bash
$ jkd_security3_injector1 selfer1_example.exe
```
![대상 검증 화면](images/jkd_security3_injector1-tgtval.png)
