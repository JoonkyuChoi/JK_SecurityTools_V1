# 3급 보안 확인툴 (jkd_security3_checker1.exe)

대상 파일(SCF)의 간단한 검증을 위한, 공개용 3급 보안툴이다.</br>
기업용인 `2급 보안 분석툴`은 검증 정보가 상세하여, 보안 상 공개할 수 없기 때문에, 대체용으로 개발한 것이다.
> 명칭
  - 확인툴
> 내장 기능
  - 자체 SCB 검증 (초간단 검증)
  - 대상 SCB 검증 (초간단 검증)
> 특징
  - 공개용 대상검증이 목적으로, 대상의 남은 [사용:기간/횟수]를 알 수 있다.
  - 출력정보에 보안적인 정보는 포함되지 않는다.


## 자체 검증 화면
패스워드 없이, 무기한 사용으로 설정된 홍보용 버전이다.</br>
```bash
$ jkd_security3_checker1
```
![자체 검증 화면](images/jkd_security3_checker1.png)

## 대상 검증 화면
패스워드 지정, 사용횟수 1회로 주입된 대상(selfer1_example.exe)을 검증한 장면이다.</br>
```bash
$ jkd_security3_checker1 selfer1_example.exe
```
![대상 검증 화면](images/jkd_security3_checker1-tgtval.png)
